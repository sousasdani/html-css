Capítulo 1

Capítulo 1.1 - CSS INLINE

A primeira forma de usar o CSS é o inline, mas não é recomendável e o professor até mesmo disse que ele é bem fraquinho..

Capítulo 1.2 - Código poluído

Inline significa que o CSS é feito na mesma linha que o HTML, mas isso não significa que é bonito, considerando que torna o código poluído.

Capítulo 2

Capítulo 2.1 - Organização

Enquanto aqui você luta para colocar todo estilo em cada linha de código que voc~e digita, criar um arquivo próprio limita essa visão poluída e deixa tudo mais bonito. Ou seja, o código vai ficar mais fácil de entender e mais organizado..
