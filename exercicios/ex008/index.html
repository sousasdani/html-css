Texto marcado

Podemos criar um texto marcado com a tag <mark>

Quando vamos fazer algo pontual e de maneira específica, eu posso trazer uma função CSS para o próprio HTML, sem necessariamente abrir outro arquivo, exemplo:

Resumindo, se eu quiser mudar a cor padrão do <mark>, bata apenas colocar style = background-color logo depois de mark junto de um background color.

Deve-se levar em conta que isso só vai funcionar se for para uma unica tag. Caso você tenha inúmeras tag <mark> e quer personalizar as cores de todas elas, existe uma forma melhor de agrupar elas do que fazer uma por uma. Nesse caso,só é ir abaixo da tag title que fica lá no início e abrir uma tag chamada style, dentro dessa tag, menciono o mark e só então aciono o background color

Exemplo:

<style>
mark {
background-color
}
</style>
